👊 Fist Bump (ฟังก์ชันสองคน)
================================

ท่า: กำหมัดสองกำ (สองคน) ชนกันกลางจอ → เล่นวิดีโอในโฟลเดอร์นี้

วางไฟล์วิดีโอชื่อ:
    fist bump.mp4

ใส่ได้หลายคลิป (จะเล่นต่อคิวให้) โดยแก้รายการในไฟล์ app.js ตรง:
    const FISTBUMP = { videos: [ ... ] };

หมายเหตุ: ถ้ายังไม่มีไฟล์วิดีโอ ระบบจะรันได้ปกติ แค่ทำท่าแล้วจะไม่เล่นอะไร
ทดสอบได้ด้วยการกดปุ่มเลข 9 บนคีย์บอร์ด
