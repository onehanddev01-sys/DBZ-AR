🌀 Double Kamehameha (ฟังก์ชันสองคน)
========================================

ท่า: ฝ่ามือเปิดของสองคนอยู่ใกล้กันกลางจอ (ปล่อยพร้อมกัน) → เล่นวิดีโอในโฟลเดอร์นี้

วางไฟล์วิดีโอชื่อ:
    double kamehameha.mp4

ใส่ได้หลายคลิป (จะเล่นต่อคิวให้) โดยแก้รายการในไฟล์ app.js ตรง:
    const KAMEHAMEHA = { videos: [ ... ] };

หมายเหตุ: ถ้ายังไม่มีไฟล์วิดีโอ ระบบจะรันได้ปกติ แค่ทำท่าแล้วจะไม่เล่นอะไร
ทดสอบได้ด้วยการกดปุ่มเลข 0 บนคีย์บอร์ด
