🐉 Father-Son Kamehameha (ฟังก์ชันหลายคน)
================================

ท่า: ฝ่ามือเปิด "สามมือเรียงกันเป็นแนวตั้ง" → เล่นวิดีโอในโฟลเดอร์นี้

วางไฟล์วิดีโอชื่อ:
    father son kamehameha.mp4

ใส่ได้หลายคลิป (จะเล่นต่อคิวให้) โดยแก้รายการในไฟล์ app.js ตรง:
    const FATHERSON = { videos: [ ... ] };

หมายเหตุ: ถ้ายังไม่มีไฟล์วิดีโอ ระบบจะรันได้ปกติ แค่ทำท่าแล้วจะไม่เล่นอะไร
ทดสอบได้ด้วยการกดปุ่ม - (ขีดกลาง) บนคีย์บอร์ด
